# Frontend React - CRUD Productos e Inventario

Frontend migrado a React + Vite para el proyecto `05-crud-full-stact-react`.

## Requisitos

- Node.js 18+
- Backend Express ejecutándose (por defecto en `http://localhost:9874`)

## Variables de entorno

Este proyecto usa la variable:

- `VITE_API_BASE_URL`: URL base del backend.

Archivos incluidos:

- `.env.example` (plantilla)
- `.env.local` (configuración local lista para usar)

Valor por defecto configurado:

`VITE_API_BASE_URL=http://localhost:9874`

## Ejecutar en desarrollo

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
```
